# Customer-Revenue-and-Rental-Behavior-Analysis

## Project Overview
This project analyzes global rental revenue and customer behavior to identify high-performing regions, renter segments, and content preferences. The analysis combines SQL-based data preparation with an interactive Tableau dashboard for clear business insights.

---

## Business Objective
The goal of this project is to understand:
- Which regions generate the most efficient revenue
- How renter segments differ in payment behavior
- Which customer groups contribute the highest value
- How preferences vary across countries and continents

---

## Data & Tools
- **Data Source:** Rental data warehouse (customer, rentals, payments, geography)
- **SQL:** Data extraction, joins, aggregation
- **Tableau:** Interactive dashboard and visual storytelling

---

## Key Insights
- Customers from Asia, especially India and China, show high revenue efficiency
- VIP renters generate the highest revenue per rental despite lower rental counts
- Asia contributes the largest share of total revenue
- Customer preferences vary significantly by country and category
